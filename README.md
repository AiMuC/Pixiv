# Pixiv
使用PHP获取Pixiv图片<br>

# 使用方法
1.域名/目录/api.php?type=getimg&url=pixiv图片地址<br>
2.域名/目录/api.php?type=getimgs&id=图片ID&page=图片页码 默认为0<br>
示例1 http://127.0.0.1/pixiv/api.php?type=getimg&url=https://i.pximg.net/img-original/img/2021/02/09/01/12/48/87632329_p0.png<br>
示例2 http://127.0.0.1/pixiv/api.php?type=getimgs&id=87632329&page=0<br>

# 注意事项

# 需要在"魔法环境内使用" 推荐境外虚拟空间 香港 新加坡 等……
